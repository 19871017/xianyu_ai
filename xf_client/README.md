# 闲鱼AI助手

## 功能
- 🔍 关键词搜索采集闲鱼商品
- 🏠 主页链接采集
- ✍️ AI文案优化改写（DeepSeek/OpenAI）
- 📦 批量上架闲鱼
- 💰 批量降价管理
- 📊 Excel导出
- 🔑 License授权管理

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python main.py
```

## 打包

```bash
build_windows.bat
```

## 配置

- 服务器地址在 `config.py` 中修改
- AI API Key 在设置页面配置
- License Key 在设置页面激活
